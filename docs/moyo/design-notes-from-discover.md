## 검색 결과 화면 구조 (사용자 자발적 제안)

- 상단: thumbs up 한 영상 3개 + "더보기"
- divider
- 하단: 최신순 영상들

## 영상 카드 표시 메타

- 기본: 썸네일·제목·채널 (유튜브 표준)
- 본인 기록: 시도 횟수, 마지막 시도일, 별점 평균 → **카드에 직접 노출** (탭 안 해도 보이는 위치)

## thumbs down 영상 시각 표현

- 검색 결과 최신순 섹션에 "흐리게" 표시 (완전 숨김 아님)
- "이미 시도했는데 별로였음" 인지 가능하도록

---

## 미정 디테일 (DESIGN 페이즈에서 결정)

- thumbs up 영상 카운트 0~2개일 때 상단 섹션 처리 (섹션 자체 미표시는 PRD 로직으로 결정됨; 그 외 시각 처리는 DESIGN 영역)
- "더보기" 인터랙션 (인라인 확장 / 별도 화면)
- description / 고정댓글 노출 위치 (영상 상세 화면 내 어디에 배치할지)
- description max length 처리 형태 (접기 버튼, 스크롤 등)
- 메뉴 페이지 레이아웃
- 시도 기록 입력 폼 형태
